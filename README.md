# ui-test-automation-playground 

This repository contains automated UI tests for the [UI Testing Playground](http://uitestingplayground.com) website, using Cypress.


## Overview
This project is designed to practice automated UI testing for various pages of the UI Testing Playground website.  

## Pages Tested
-Click
-DisableInput
-Dynamic-ID
-HiddenLayers
-LoadDelay
-NonBreakingSpace
-ajax
-classAttribute
-clientdealy
-inputText
-sampleapp
